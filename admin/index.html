Basis-Konfiguration

Marke, Land und Intervall festlegen. Danach unten den OAuth2-Login starten.

Stellantis Login (OAuth2)

Seit Anfang 2024 erfordert Stellantis eine Browser-Anmeldung. Klicke auf „Auth-URL erzeugen", öffne den Link im Browser, melde dich mit deinem Stellantis-Account an und kopiere dann den Code aus der Weiterleitungs-URL in das Feld unten.

Schritt 1 – Auth-URL erzeugen

Speichere zuerst Brand und Ländercode oben, dann klicke den Button.

🌐 Im Browser öffnen

Nach dem Login wird der Browser auf eine URL wie mymap://oauth2redirect/de?code=DEIN_CODE&state=... oder eine Fehlerseite weitergeleitet. Kopiere den Wert des code=-Parameters.
